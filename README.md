# blacklist
IP Blacklister (Debian)

#Overview

Escanea el log de acceso y busca intentos fraudulentos de acceder por root o con cuentas inexistentes.
Luego banea las IPs. 

Utiliza firewalld (sudo apt-get install firewalld)
